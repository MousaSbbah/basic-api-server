# basic-api-server


**V1.0**

## About <a name = "about"></a>

Basic 2 APIs server which supported 
`GET`
`POST` 
`DELETE`
`PUT`

By Rout At `/student` and `/animal`

## Deployment 
This project use Heroku to deployment

API Live Url: **https://mousa-basic-api-server.herokuapp.com/**
